P2P File Sharing System

Simplified BitTorrent-style P2P file sharing with a UDP tracker, TCP transfers, and SHA-256 integrity checks.

Python | Sockets | UDP | TCP | Threading | SHA-256 | Tkinter

GitHub

Features

  • Tracker registers seeders + file hashes
  • Peers discover seeders via UDP
  • File transfers via TCP (chunked)
  • SHA-256 verification after download (auto + manual)

How it works

  1. Seeder registers file + hash with tracker (UDP)
  2. Leecher requests seeders list (UDP)
  3. Leecher downloads from chosen peer (TCP)
  4. Hash verifies integrity; if fail â†’ try next seeder
  5. Complete file download with progress bar (Tkinter)

Challenges

  • Handling peers going offline (heartbeat + timeout)
  • Avoiding blocking during transfers (multithreading)
  • Verification failure handling (delete corrupted file + fallback seeder)

Improvements

  • Add encryption/authentication
  • Support wide-area networks (NAT traversal)
  • Better chunking/resume downloads (more BitTorrent-like)
