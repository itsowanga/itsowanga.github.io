# Portfolio Website

Personal portfolio site for Sowanga Mbane. It highlights education, skills,
and featured projects, with links to social profiles and contact options.

## Pages

- `index.html` - Home page with overview, featured projects, and contact links.
- `projects.html` - Full projects list.
- `p2p.html` - Peer-to-peer file sharing project details.
- `codepulse.html` - CodePulse project details.
- `farmyield.html` - FarmYield project details.
- `contact.html` - Contact page.

## View Locally

1. Open `index.html` in your browser.
2. Navigate using the links in the site header and page content.

## Assets

- `style.css` - Shared styles.
- `assets/` - Images, logo, and resume.
