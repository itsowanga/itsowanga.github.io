FarmYield Predictor - Decision Support Tool

Offline-first yield prediction + risk alerts for smallholder maize farmers using a lightweight regression model.

Python | C++ | Machine Learning

GitHub

Features

  • Offline-first yield prediction
  • Risk alerts for smallholder maize farmers
  • Lightweight regression model
  • Simple UI designed for low-spec devices

How it works

  1. Farmer enters field data and seasonal inputs
  2. Model estimates expected yield for the season
  3. Risk rules flag low-yield scenarios early
  4. Results display with simple recommendations

Challenges

  • Limited connectivity required offline-first design
  • Small datasets made model tuning difficult
  • Balancing accuracy with speed on low-spec devices

Improvements

  • Add soil sensor integrations for richer inputs
  • Support additional crops beyond maize
  • Improve explainability for prediction drivers
